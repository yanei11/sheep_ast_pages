#new(key, sym = nil, **options) ⇒ ExactMatch

To create exact match

It match if given expression == <expr>. It store matched expression to [store symbol] in the data. If store symbol is not specified, framework gives default symbol.

Examples:

E(:e, '<expr>', [store symbol], [options])

Parameters:

  • key (String)
  • sym (Symbol, nil) (defaults to: nil)
  • options (T.untyped)

Options Hash (**options):

  • :repeat (Integer) —

    Retruns same instance for specified count

  • :extract (Range) —

    To modify matched string by range

  • :at_head (Boolean) —

    Match when the expression is head of the sentence

  • :index_cond (IndexCondition) —

    Additional condition to match arbeitary index of sentene
