Overview

This module is part of a private API. You should avoid using this module if possible, as it may be removed or be changed in the future.

utility function to handle Index of tokenized sentence. This utility enable to compute expression from specified index.

e.g. given tokenized sentence is [[a, b, c, d, e, n], [f, g, h, i, n]] and given that you have matched expression d, then this utility allow user to give index 4 to specify 'h' word. It can handle multiline situation and defaultly it ignores 'n' newline indication
